# 2.3 反转链表

[206. 反转链表](https://leetcode.cn/problems/reverse-linked-list/)

```cpp
class Solution {
public:
    ListNode* reverseList(ListNode* head) {
        ListNode* p = nullptr;
        while(head!=nullptr){
            ListNode* tmp = head->next;
            head->next = p;
            p = head;
            head = tmp;
        }
        return p;
    }
};
```
