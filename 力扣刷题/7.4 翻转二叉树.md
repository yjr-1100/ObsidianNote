# 7.4 翻转二叉树

递归的方法：前序遍历翻转

```cpp
class Solution {
public:
    void invert(TreeNode* node){
        if(node==nullptr) return ;
        if(node->left==nullptr&&node->right==nullptr) return;
        TreeNode* tmp = node->left;
        node->left = node->right;
        node->right = tmp;
        invert(node->left);
        invert(node->right);
    }
    TreeNode* invertTree(TreeNode* root) {
        invert(root);
        return root;
    }
};
```
